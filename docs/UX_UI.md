## 🎨 PALETA DE COLORES

### Versión Europa/América (Default)

| Nombre | Hex | Uso |
|--------|-----|-----|
| **Primary** | `#1e3a5f` | Headers, botones principales, links |
| **Secondary** | `#c9a227` | Acentos, CTAs secundarios, destacados |
| **Accent** | `#2e7d32` | Éxito, badges, elementos positivos |
| **Text** | `#2c3e50` | Texto principal |
| **Light** | `#f8f9fa` | Fondos claros, secciones alternadas |
| **White** | `#ffffff` | Fondos, cards |
| **Dark** | `#1a1a2e` | Footer, elementos oscuros |

### Versión China (para :lang(zh))

| Nombre | Hex | Uso |
|--------|-----|-----|
| **Primary** | `#8b0000` | Rojo - suerte y prosperidad |
| **Secondary** | `#ffd700` | Dorado - riqueza |
| **Accent** | `#1e3a5f` | Azul - confianza |

### CSS para cambio de paleta:

```scss
:root {
    --color-primary: #1e3a5f;
    --color-secondary: #c9a227;
    --color-accent: #2e7d32;
}

:lang(zh) {
    --color-primary: #8b0000;
    --color-secondary: #ffd700;
    --color-accent: #1e3a5f;
}

.btn-primary {
    background-color: var(--color-primary);
}
```

## 🔤 TIPOGRAFÍA

### Fuentes

| Fuente | Uso | Peso | Fallback |
|--------|-----|------|----------|
| **Inter** | Latino (ES, EN, PT, FR) | 400, 500, 600, 700 | system-ui, sans-serif |
| **Noto Sans SC** | Chino (ZH) | 400, 500, 700 | Microsoft YaHei, SimHei |

### Escala Tipográfica

| Elemento | Tamaño | Peso | Line Height |
|----------|--------|------|-------------|
| H1 | 3rem (48px) | 700 | 1.2 |
| H2 | 2.25rem (36px) | 700 | 1.25 |
| H3 | 1.5rem (24px) | 600 | 1.3 |
| H4 | 1.25rem (20px) | 600 | 1.4 |
| Body | 1rem (16px) | 400 | 1.6 |
| Small | 0.875rem (14px) | 400 | 1.5 |
| Caption | 0.75rem (12px) | 400 | 1.4 |

### CSS Tipografía

```scss
// Fuentes
$font-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
$font-chinese: 'Noto Sans SC', 'Microsoft YaHei', 'SimHei', sans-serif;

body {
    font-family: $font-primary;
    font-size: 1rem;
    line-height: 1.6;
    color: $text-color;
    
    &:lang(zh) {
        font-family: $font-chinese;
    }
}

h1, h2, h3, h4, h5, h6 {
    font-weight: 700;
    line-height: 1.25;
    color: $primary;
}
```

## 📐 MOCKUPS 5 PÁGINAS PRINCIPALES

### 1. INICIO

**Estructura:**
- Header sticky con navegación + selector idioma
- Hero fullscreen con fondo oscuro + overlay
- 3 cards de servicios en grid
- Lista de beneficios con iconos
- 4 estadísticas con animación counter
- 2 testimonios en slider
- CTA final con fondo de color
- Footer 4 columnas

**Elementos distintivos:**
- Scroll suave entre secciones
- Animaciones fade-in al scroll
- Botón WhatsApp flotante (solo móvil)

### 2. QUIÉNES SOMOS

**Estructura:**
- Hero interno con breadcrumb
- Misión + Visión en 2 columnas
- 4 valores en grid con iconos
- Equipo en cards circulares
- CTA a contacto

### 3. NUESTRO PROCESO

**Estructura:**
- Timeline vertical con 5 pasos
- Cada paso: número, título, duración, actividades
- Iconos animados
- Indicador "Primera consulta GRATUITA"

### 4. CASOS DE ÉXITO

**Estructura:**
- Filtros por tipo (tabs)
- Grid de cards de casos
- Card: imagen, título, extracto, link
- Paginación

### 5. CONTACTO

**Estructura:**
- 2 columnas: Formulario + Info
- Formulario con validación
- Mapa Google embed
- Horarios y redes sociales
